## <a id="english">🇺🇸 English</a>

### 📋 Overview
A simple full-stack task management application built with React and Express.js. Perfect for learning modern web development concepts.

### ✨ Features
- ✅ Create, edit, and delete tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Real-time data persistence
- ✅ Clean, minimalist UI
- ✅ Full-stack architecture
- ✅ RESTful API design

### 🚀 Quick Start

#### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

#### Installation
```bash
# Clone the repository
git clone https://github.com/peaske/demo-task-app.git
cd demo-task-app

# Install frontend dependencies
npm install

# Install backend dependencies
cd backend
npm install
cd ..
```

#### Running the Application
```bash
# Terminal 1: Start backend server
cd backend
npm start
# Runs on http://localhost:3001

# Terminal 2: Start frontend development server
npm run dev
# Runs on http://localhost:5173
```

### 🛠️ Tech Stack
- **Frontend**: React 18, Vite
- **Backend**: Node.js, Express.js
- **Storage**: JSON files
- **Styling**: Vanilla CSS

### 📡 API Endpoints
- `GET /api/tasks` - Get all tasks
- `POST /api/tasks` - Create new task
- `PUT /api/tasks/:id` - Update task
- `DELETE /api/tasks/:id` - Delete task
